Este sitio muestra el resultado de una validación. Para máxima seguridad, valida en un servidor.

Cómo funciona

  1. El QR dirige a esta página con un parámetro code.
  2. La página consulta un servicio de verificación (o usa una demo local).
  3. Si el código es válido, muestra “Documento original” y los metadatos.

Si quieres, puedo adaptarlo para tu flujo: por ejemplo, validar también el ID de documento, mostrar un PDF, o integrar firma digital.
